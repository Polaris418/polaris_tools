## 架构亮点

### 1. 安全性
- Token 使用 SHA-256 哈希存储，不存明文
- Token 一次性使用，用完即废
- 支持 Token 过期和自动清理

### 2. 可靠性
- 抑制列表自动处理退信和投诉
- 软退信计数机制（3 次后抑制）
- 发送前检查抑制列表
- 队列系统支持失败重试（指数退避）

### 3. 可扩展性
- 队列系统支持优先级
- 多线程并发处理
- 模板系统支持多语言
- 监控系统支持自定义告警阈值

### 4. 可维护性
- 模板集中管理，支持在线编辑
- 完整的审计日志
- 监控仪表板实时展示指标
- 订阅偏好灵活配置
